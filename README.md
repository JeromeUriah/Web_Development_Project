-- Web Development Project --

A multi-page web application built to demonstrate modern front-end development principles, structured project iteration, and Progressive Web App (PWA) implementation.
Developed as part of DECO7140, this project showcases practical skills in responsive design, client-side scripting, and staged development workflows.

-- Project Overview -- 

This project was designed and implemented across multiple development phases, focusing on:
* Structured multi-page architecture
* Responsive layout and UI styling
* Client-side interactivity using JavaScript
* Progressive Web App setup (manifest, service worker, installability)
* Basic backend integration testing (JSP / PHP endpoints)

The repository reflects an iterative development process, separating implementation into clear work phases to simulate real-world project progression.

-- Key Features -- 

* Multi-page navigation (index.html, journal.html)
* Responsive UI design using CSS
* JavaScript-driven interactivity
* Progressive Web App support (offline readiness and install configuration)
* Modular project organization across development stages
* Backend environment testing files (test.jsp, test.php)

-- Technologies used -- 

HTML5
CSS3
JavaScript (Vanilla JS)
Progressive Web App APIs
JSP / PHP (basic integration testing)
